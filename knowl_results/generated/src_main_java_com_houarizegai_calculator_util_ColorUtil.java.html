Function name: hex2Color

Input:

colorHex: A string representing a hexadecimal color value.

Output:

A Color object representing the specified hexadecimal color value.

Description:

This JavaScript function takes a hexadecimal color value as input and converts it into a Color object. The function uses the Java 8 Optional class to handle the possibility of the input being null.

If the input is not null, the function extracts the red, green, and blue components from the hexadecimal value using the substring method. These components are converted from hexadecimal to decimal using the Integer.valueOf method with the base 16. Then, a new Color object is created using these decimal values for the red, green, and blue components. Finally, the Color object is returned as the result of the function. If the input is null, the function returns null.
