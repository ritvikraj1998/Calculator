Function name: getThemesAsMap

Input:

List of themes

Output:

A Map of String and Theme objects

Description:

This function takes a list of themes as input and returns a Map of String and Theme objects. The input list is streamed, which allows for a sequence of operations to be performed on each element of the list. The stream is collected into a Map using the Collectors.toMap method. The toMap method requires two arguments - a key mapper (Theme::getName) and a value mapper (Function.identity()). The key mapper maps each element of the input list to its name property, which is used as the key in the resulting Map. The value mapper maps each element of the input list to itself. Finally, the function returns the Map as the output.
