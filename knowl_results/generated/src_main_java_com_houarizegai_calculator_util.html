Description

This folder contains utility files related to colors.

Contents:

ColorUtil.java: The `ColorUtil` class provides utility methods for colors. It has a constructor and a static method `hex2Color(String colorHex)` that converts a hexadecimal color code to a `Color` object. It handles null input and returns null if the input is null. Otherwise, it converts the hex code to integer values of red, green, and blue components and creates a new `Color` object. There are no other instance methods in this class.
