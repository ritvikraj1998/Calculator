Function name: calculate

Input:

firstNumber: This is the first number for calculation.

secondNumber: This is the second number for calculation.

operator: This is the operator that specifies the operation to be performed. It can be one of the following: + (addition), - (subtraction), * (multiplication), / (division), % (modulus), or ^ (exponentiation).

Output:

result: This is the calculated result of the operation.

Description:

This function takes in two numbers, firstNumber and secondNumber, along with an operator to perform a calculation. It uses a switch statement to determine the operation based on the operator provided.

If the operator is '+', it adds firstNumber and secondNumber.

If the operator is '-', it subtracts secondNumber from firstNumber.

If the operator is '*', it multiplies firstNumber and secondNumber.

If the operator is '/', it divides firstNumber by secondNumber.

If the operator is '%', it calculates the modulus of firstNumber divided by secondNumber.

If the operator is '^', it calculates the exponentiation of firstNumber to the power of secondNumber using the Math.pow() function.

If the operator is none of the above, it returns the value of secondNumber.

The calculated result is then returned as the output of the function.

Output:

Description:

This function initializes a theme selector in the user interface. It creates a combo box with theme names as options and sets up an item listener to handle theme selection events.

First, it creates the combo box by calling the function createComboBox with the keys of themesMap as the options, a width of 230 pixels, a height of 30 pixels, and a label of "Theme".

Then, it adds an item listener to the combo box using a lambda expression. The lambda expression checks if the event state change is "selected", and if so, it retrieves the selected theme name and calls the function applyTheme with the corresponding theme value from themesMap.

Next, it checks if there is at least one entry in themesMap. If so, it applies the first theme in the map by calling applyTheme with the value of the first entry.

Function name: applyTheme

Input:

theme : An object of the Theme class that contains various properties for customizing the application's appearance.

Output:

No specific output. This function applies the provided theme to various UI elements of the application.

Description:

This function is responsible for applying the provided theme to the different UI elements of the application. It takes a Theme object as an input and uses its properties to set the background, foreground, and color of various components.

The `applyTheme` function is called with a Theme object as `theme`.

First, it sets the background color of the `window.getContentPane()` to the corresponding color specified by `theme.getApplicationBackground()`. This function `hex2Color(theme.getApplicationBackground())` converts the hexadecimal color code to the actual color value.

Next, it sets the foreground (text color) of various UI components such as `comboCalculatorType`, `comboTheme`, `inputScreen`, `btn0`, `btn1`, ..., `btnEqual` using the same `hex2Color(theme.getTextColor())` function.

Similarly, it sets the background color of various buttons (`btn0`, `btn1`, ..., `btnEqual`) and other UI components such as `comboCalculatorType`, `comboTheme`, and `inputScreen` to the respective color values specified by the `theme` object's properties (e.g., `theme.getNumbersBackground()`, `theme.getOperatorBackground()`, `theme.getBtnEqualBackground()`, etc.) using the `hex2Color()` function.
